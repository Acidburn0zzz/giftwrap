giftwrap
========

Usage
=====

    $ pip install giftwrap
    $ giftwrap -h

Dependencies
============

* `Vagrant`
* `fpm`

Development
===========

    $ git clone git@github.com:cloudcadre/giftwrap.git
    $ vagrant up

Testing
=======

    $ make test
